Instructions

  • Change the sliders, in the left menu, to see different spirals and patterns emerge
  • Alternatively enter a specific value in the text fields
  • The right menu is a colorpicker - change the values for hue, saturation and lightness to make a color
  • Once you have selected a color press either set color A or set color B - The spiral will transistion between color A and color B
  • Select a preset from the dropdown and and click 'load preset' to see premade spirals
  • Click the 'Randomize' button to see spirals with randomised settings
  • Right click on the spiral to save it as a picture
